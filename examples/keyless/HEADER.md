# Keyless example

The total-automated-keyless-deployment showcase: shared keys disabled, identity-authenticated
storage, and the FastAPI package pushed by the pipeline's dedicated deploy stage with a fresh
OIDC login. No storage keys, no SAS, anywhere. The environment comes from the Terraform workspace
(`terraform.workspace`), not a variable.
