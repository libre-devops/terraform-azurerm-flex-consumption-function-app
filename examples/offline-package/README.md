# Offline package example

The egress-blocked pattern: dependencies are vendored on the runner (pip install --target
.python_packages/lib/site-packages), the package ships byte-identical, and nothing is built or
downloaded server-side. Mandatory when the app cannot reach PyPI. The environment comes from the
Terraform workspace (`terraform.workspace`), not a variable.

[![Terraform Registry](https://img.shields.io/badge/registry-libre--devops-7B42BC?logo=terraform&logoColor=white)](https://registry.terraform.io/namespaces/libre-devops)
